# Medical Appointment Booking

Medical Appointment Booking is a healthcare frontend application developed using React.js.

## Features

- User Login
- User Registration
- Appointment Booking
- Doctor Search
- Reviews and Ratings
- Profile Management

## Technologies Used

- React.js
- React Router DOM
- CSS

## Setup Instructions

1. Clone the repository

git clone https://github.com/mallurisivacharan/grihf-frontend_capstone_starter_code.git

2. Navigate to the project folder

cd grihf-frontend_capstone_starter_code

3. Install dependencies

npm install

4. Start the development server

npm start

5. Open browser and visit

http://localhost:3000# Medical Appointment Booking
